# Hashmaps

A *hash map* allows you to associate a value with a particular key.
You may also know this by the names [*unordered map* in C++](https://en.cppreference.com/w/cpp/container/unordered_map),
[*dictionary* in Python](https://docs.python.org/3/tutorial/datastructures.html#dictionaries) or an *associative array* in other languages.

This is the other data structure that we've been talking about before, when
talking about Vecs.

# 哈希映射

哈希映射允许你将一个值与一个特定的键关联起来。你可能也知道它们被称为 [C++ 中的 *unordered map*](https://en.cppreference.com/w/cpp/container/unordered_map)，[Python 中的 *dictionary*](https://docs.python.org/3/tutorial/datastructures.html#dictionaries) 或其他语言中的 *associative array*。

这就是我们之前讨论的另一种数据结构，和 Vec 一起讨论时提到的。
